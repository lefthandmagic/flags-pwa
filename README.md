# Flags (PWA)

Guess the country from the flag. Built to iterate via WhatsApp with Luna; port to iOS (Expo) later when happy.

## Run locally

```bash
cd apps/flags-pwa
npx --yes serve -l 5173
```

Open `http://localhost:5173` (or the Mac mini LAN IP from your iPhone on the same Wi‑Fi).

## iPhone install feel

Safari → Share → **Add to Home Screen**.

## Feedback loop

Message Luna on WhatsApp with what felt wrong (too easy, wrong names, UI) and we patch.
